# 🚀 Запуск FinanceAI в Telegram — пошаговая инструкция

## ШАГ 1. Загрузить код на GitHub

### 1.1 Установить Git (если нет)
- Windows: скачать с https://git-scm.com → установить
- Mac: в терминале `git --version` → если нет, предложит установить

### 1.2 Создать репозиторий на GitHub
1. Зайдите на https://github.com
2. Нажмите зелёную кнопку **New** (или нажмите +→ New repository)
3. Repository name: `finance-tma`
4. Оставьте **Public** (или Private)
5. Нажмите **Create repository**
6. На следующей странице скопируйте ссылку — она вида `https://github.com/ВАШЕ_ИМЯ/finance-tma.git`

### 1.3 Загрузить файлы
Откройте терминал в папке проекта и выполните по очереди:

```bash
git init
git add .
git commit -m "FinanceAI Telegram Mini App"
git branch -M main
git remote add origin https://github.com/ВАШЕ_ИМЯ/finance-tma.git
git push -u origin main
```

Если попросит войти в GitHub — войдите через браузер когда предложит.

---

## ШАГ 2. Опубликовать на Vercel

1. Зайдите на https://vercel.com
2. Нажмите **Add New → Project**
3. Найдите ваш репозиторий `finance-tma` → нажмите **Import**
4. Framework Preset: **Vite** (выберите из списка)
5. Нажмите **Deploy** — подождите 1-2 минуты

✅ Готово! Vercel выдаст адрес вида `finance-tma-xxx.vercel.app` — это и есть URL вашего приложения. Запишите его.

### 2.1 Добавить переменные окружения (токен бота)
1. В Vercel откройте проект → **Settings → Environment Variables**
2. Добавьте:
   - Name: `BOT_TOKEN`  Value: `ВАШ_ТОКЕН_БОТА`
   - Name: `APP_URL`    Value: `https://finance-tma-xxx.vercel.app`
3. Нажмите **Save**
4. Зайдите в **Deployments** → последний деплой → **Redeploy** (чтобы переменные применились)

---

## ШАГ 3. Настроить webhook бота

Откройте браузер и перейдите по этой ссылке (замените TOKEN и URL):

```
https://api.telegram.org/botВАШ_ТОКЕН/setWebhook?url=https://finance-tma-xxx.vercel.app/api/webhook
```

Пример:
```
https://api.telegram.org/bot8782203254:AAG.../setWebhook?url=https://finance-tma.vercel.app/api/webhook
```

Браузер покажет: `{"ok":true,"result":true}` — это значит бот подключён! ✅

---

## ШАГ 4. Добавить кнопку приложения в бот

1. Откройте Telegram → найдите **@BotFather**
2. Напишите `/setmenubutton`
3. BotFather спросит бота — выберите вашего
4. Напишите URL: `https://finance-tma-xxx.vercel.app`
5. Напишите текст кнопки: `📊 FinanceAI`

Теперь внизу у вашего бота появится синяя кнопка «📊 FinanceAI».

---

## ШАГ 5. Проверить

1. Откройте вашего бота в Telegram
2. Нажмите кнопку `📊 FinanceAI` внизу
3. Откроется приложение прямо внутри Telegram!

Или напишите боту `/start` — он пришлёт кнопку для открытия.

---

## Как обновить приложение после изменений

Просто выполните в терминале:
```bash
git add .
git commit -m "обновление"
git push
```

Vercel автоматически подхватит изменения и опубликует за ~1 минуту. Никакой ручной пересборки!

---

## Возможные проблемы

**«Application Error» в Telegram** — зайдите на Vercel, проверьте Build Logs на ошибки.

**Бот не отвечает** — проверьте что webhook установлен: 
`https://api.telegram.org/botТОКЕН/getWebhookInfo`

**Приложение открывается в браузере а не в Telegram** — это нормально при разработке. Внутри Telegram всё работает.
